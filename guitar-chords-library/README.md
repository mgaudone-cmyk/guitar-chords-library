# 🎸 Matt's Tool — Community Song Library

A shared collection of guitar chord sheets for use with [Matt's Tool](https://mgaudone-cmyk.github.io/guitar-chords).

## How to use

Open Matt's Tool, click the **🌐 Community** tab, and browse or search songs. Click any song to add it to your library.

## How to contribute a song

1. Click **Add file** → **Create new file** above
2. Name it: `songs/Song_Title_Artist.txt`
3. Use this format:

```
Song Title
Artist Name
Key: X | Capo: X (optional)

[Section]
Am              G
Chord line above the lyric
The lyric line goes here
```

4. Click **Commit new file**

Your song will appear in the Community tab for everyone automatically.

## Song naming convention

`Song_Title_Artist.txt` — use underscores for spaces, no special characters.

Examples:
- `Hotel_California_Eagles.txt`
- `Wonderwall_Oasis.txt`
- `Sweet_Home_Chicago_Blues.txt`
